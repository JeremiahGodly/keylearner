# Preview Project Workflow

This workflow describes how to start the local Next.js development server to preview and test the web application.

## Prerequisites

- Ensure dependencies are installed and Node.js v20 is active:
  ```bash
  source ~/.nvm/nvm.sh && nvm use 20
  ```

## Steps

1. **Start Development Server**:
   Launch the development server on the configured port 9002:
   ```bash
   npm run dev
   ```

2. **Open the App**:
   Navigate to the following URL in your web browser:
   [http://localhost:9002](http://localhost:9002)
