# **App Name**: KeyLearner

## Core Features:

- Keyboard Fundamentals: Interactive introduction to keyboard layout, basic posture, and finger placement.
- Essential Music Theory: Simplified lessons on scales, chords, and rhythm, tailored for beginners.
- Interactive Song Tutorials: Step-by-step tutorials for playing popular songs across various genres.
- Personalized Learning Path: An AI tool assesses user skills and suggests a tailored learning path through the available lessons and exercises.
- Progress Tracker: A system for users to monitor their learning progress, track achievements, and identify areas for improvement.
- Resource Library: A repository of practice exercises, sheet music downloads, and video demonstrations to supplement learning.
- Dark Mode: An option to switch to a dark color scheme, reducing eye strain during long practice sessions.

## Style Guidelines:

- Primary color: A deep blue (#3F51B5) evoking focus and concentration.
- Background color: A very dark blue (#1A237E), nearly black, to focus attention on the learning content and support the dark mode option.
- Accent color: A vibrant purple (#9C27B0) to highlight interactive elements and important information.
- Body and headline font: 'Inter' sans-serif for clear, readable text.
- Use clean, minimalist icons representing musical concepts and functions.
- Divide content into short, digestible modules, each with clear objectives and progress indicators.
- Subtle animations and transitions to provide feedback and enhance interactivity.
