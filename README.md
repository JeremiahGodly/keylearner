# KeyLearner

An interactive piano and keyboard learning application built with vanilla HTML, CSS, and JavaScript.

## Features

- **No Server Needed**: Runs completely on the client side.
- **Interactive Piano**: Play notes by clicking keys or using your computer keyboard (`A` `S` `D` `F` `G` `H` `J` for white keys, `W` `E` `T` `Y` `U` for black keys).
- **Web Audio Synthesis**: Note sounds are dynamically synthesized using the browser's built-in Web Audio API.
- **Dynamic YouTube Video Classes**: Select lessons in English or Malayalam. Content filters automatically based on your language choice.
- **Song Practice Mode**: Follow highlighted keys to play popular songs like "Twinkle Twinkle Little Star" or "Ode to Joy" note-by-note.
- **Weekly Progress Tracker**: Track mock practice times.
- **Theme Switcher**: Smooth transition between light and dark modes with premium gradient colors.
